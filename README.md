This project is an interactive web visualization of my personal screen time data built using **D3.js**. It shows which apps I spent the most time on and lets users explore the data through filters and interactive charts. I did this as part of an assignment for my information visualization class.

---

## What You Can Do

- View **Top N apps by total screen time**
- Filter by **category** and **date range**
- Adjust how many apps are shown
- **Hover** over bars to see exact usage values
- **Click** a bar to see a daily breakdown chart for that app

---

## Tech Used

- D3.js  
- HTML, CSS, JavaScript  
- GitHub Pages
